HTML Tag Reference

Other HTML Tags

Tag Name Description
<h1> to <h6> Heading 1 The <h1> to <h6> tags are used to define HTML headings.
<p> Paragraph Browsers automatically add a single blank line before and after each <p> element.
<span> Span an inline container used to mark up a part of a text, or a part of a document.
<div> Division defines a division or a section in an HTML document.
<img> Image used to embed an image in a web page.
